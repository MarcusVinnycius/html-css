Existem cinco valores de posições diferentes

  • static
  • relative
  • fixed
  • absolute
  • stikey

Os elementos são posicionados usando as propriedades top, bottom, left, right.

Position 'static'

Os elementos em html por padrão já são posicionados em static. Não são afetados pelas propriedades, top, left, right e bottom.

Position 'relative'

São afetados pelas propriedades top, left, right e bottom. O conteúdo não vai se ajustar para se encaixar no espaço deixado pelo elemento.

Position Fixed

O elemento é posicionado em relação a viewport, ou seja, ele vai permanecer no mesmo lugar da página.
